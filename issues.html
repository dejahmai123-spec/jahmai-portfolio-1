Issues

Challenges I encountered while building animations and updating code.

  1. Issue 1 — Animation timing and smoothness

    It was difficult to get the animations to run smoothly because I needed a modified code setup to match the environment I was using (One Compiler). Tweaking easing, durations, and transform properties helped improve smoothness.

  2. Issue 2 — Outdated class materials

    Many class code examples were outdated, so I had to update and remaster them, sometimes using AI to suggest modern alternatives with better browser support.

  3. Issue 3 — Hover animation accuracy

    Hover animations require precise timing and transform values. Even small miscalculations can cause jitter. I learned how to refine animation effects through testing and inspecting computed CSS.
